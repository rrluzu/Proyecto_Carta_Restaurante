Para poder utilizar el seeder de productos, 
pega todas las imágenes en la siguiente ruta:

C:\xampp\htdocs\Proyecto_Carta_Restaurante_RonyL\laravel\storage\app\public\imagenes_prueba

Las imágenes tienen que estar si o si en el directorio "imagenes_prueba",
después ejecuta el siguiente comando para utilizar el seeder de productos.

php artisan db:seed --class=ProductosSeeder
